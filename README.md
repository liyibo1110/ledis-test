# ledis

作为学习redis源码的项目，会对redis代码做逐步分析，并重写进此项目，并加入自己的注释。

将一些额外的功能暂时滞后，如内存分配，目前只会用原生的malloc、calloc来代替。

也借鉴了《redis设计与实现》，同时对作者表示感谢。
